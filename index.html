Lit Starter Samples

This page is a quick list of code samples to get you started with Lit if you're new to it!

Rendering & Templating

  • List Separators in lit-html
    • How to add commas between list items with `Array.map`.
  • Async Task / NPM Info element
    • Shows an example of an async Task controller. This one fetches npm package info and renders based on the state of the fetch.

Lifecycle

  • _getUpdateComplete()
    • How to make a parent element wait for an inner child to render.

Cross-Framework

  • <chess-board> element in React
    • Shows the <chess-board> element, implemented in LitElement (older implementation), used inside React.
  • LitElement Goo / React Hooks
